# Refeeding Syndrome One-Stop (Static)

Pure HTML/CSS/JS static site for:
- BMI, weight loss %, electrolyte % drop
- NICE adult high-risk logic
- ASPEN electrolyte-shift severity categorization
- Imminent danger flags (phos ≤ 0.6 mmol/L and/or ≥30% drop)
- Actionable care framework (calories, monitoring, thiamine, consult triggers)
- EMR copy block + Print/PDF + JSON export
- LocalStorage autosave

## Deploy on GitHub Pages
1. Create repo, add files.
2. Settings → Pages → Deploy from branch → `main` / root.
3. Open your Pages URL.

Educational decision support only; follow local protocols and clinical judgment.
